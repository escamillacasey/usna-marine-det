# Internal messaging sources

**Do not publish files in this folder.** They are gitignored and exist only to align public site copy with detachment guidance.

## Add the annual campaign order

1. Copy `campaign-order.template.md` to `campaign-order-source.md`, **or drop the campaign plan PDF** in this folder (e.g. `Campaign Plan 2025-2026.pdf`).
2. Paste or transcribe at minimum:
   - Commander's intent
   - Mission / end state (if stated separately)
   - Lines of effort or priority themes
   - Any phrasing the CO wants echoed publicly (taglines, “focus areas,” etc.)
3. Tell the agent (or run a messaging pass yourself) using `docs/MESSAGING-GUIDE.md` — update **public** pages only; never link to or upload the source document.

## Gitignored patterns

- `campaign-order-source.*`
- `*.pdf`, `*.docx`, `*.doc` in this folder

The template and this README are committed so the workflow stays documented.
