# Company mentor photos

Add one portrait per company, named sequentially:

```
company-01.jpg  → Company 1
company-02.jpg  → Company 2
…
company-36.jpg  → Company 36
```

Then update `js/company-mentors-data.js` with each mentor's `name`, `rank`, `email`, and `bio`.

Recommended image size: at least 400×500 px, portrait orientation.

Battalion mapping (6 companies each):

| Battalion | Companies |
|-----------|-----------|
| 1st | 1–6 |
| 2nd | 7–12 |
| 3rd | 13–18 |
| 4th | 19–24 |
| 5th | 25–30 |
| 6th | 31–36 |
